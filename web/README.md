# MeetConnect Web App

React web application for MeetConnect.

## Features
- Responsive web interface
- Progressive Web App (PWA)
- Real-time updates
- Map integration
- Shared components with mobile

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

App will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Tech Stack
- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Zustand (state management)
- Leaflet (maps)
