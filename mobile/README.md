# MeetConnect Mobile App

React Native mobile application built with Expo.

## Features
- Cross-platform (iOS & Android)
- Expo Router for navigation
- TypeScript for type safety
- Location services
- Push notifications
- Real-time chat

## Setup

```bash
npm install
```

## Development

```bash
# Start Metro bundler
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Tech Stack
- React Native + Expo
- TypeScript
- Expo Router (file-based routing)
- Zustand (state management)
- Socket.io (real-time)
- React Native Maps
